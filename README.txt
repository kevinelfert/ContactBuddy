Contact Buddy

Modules needed:
    -tkinter
    -email-validator

to install email-validator
    - pip install email-validator 
    - documentation for email-validator: https://pypi.org/project/email-validator/

To run Contact Buddy app
    - navigate to ContactBuddy directory
    - run: python3 contact_buddy.py
